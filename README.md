# Prozessmodell zur Wahrnehmung asynchroner Interview-Tools

Dieses Repository enthält die **interaktive 3D-Visualisierung** des Prozessmodells aus der Masterarbeit *«Ökonomische Effekte asynchroner Interview-Tools: Eine Analyse aus Arbeitgeber- und Bewerberperspektive»* an der Universität St. Gallen (HSG).

## Über das Modell

Das Modell stellt die Ergebnisse einer qualitativen Analyse von 17 Experteninterviews mit Bewerbenden, Rekrutierenden und Führungskräften dar. Es visualisiert die drei identifizierten Aggregate Dimensions sowie die sieben Second-Order Themes in einer dreistufigen Struktur:

- **Erleben und Beurteilung des Formats** (drei Themes)
- **Strukturelle und zeitliche Rahmenbedingungen des Einsatzes** (zwei Themes)
- **Technologische Einstellung & Wirkung auf das Arbeitgeberimage** (zwei Themes)

Die methodische Grundlage bildet die qualitative Inhaltsanalyse nach Gläser und Laudel (2010) in Kombination mit der Gioia-Methodik (Gioia et al., 2013).

## Live-Ansicht

Die interaktive 3D-Version ist über GitHub Pages erreichbar.

## Bedienung

| Aktion                 | Funktion                                     |
|------------------------|----------------------------------------------|
| Linke Maustaste        | Modell drehen                                |
| Rechte Maustaste       | Modell verschieben                           |
| Mausrad                | Zoomen                                       |
| Hover über Kugel       | Theme-Details anzeigen                       |
| «Ansicht zurücksetzen» | Initiale Kameraposition wiederherstellen     |
| «Bild speichern»       | Hochauflösendes PNG des aktuellen Bildes     |

Auf Touch-Geräten: Ein Finger zum Drehen, zwei Finger zum Zoomen und Verschieben, Tippen auf eine Kugel zeigt die Theme-Details.

## Technische Hinweise

Die Visualisierung wurde mit **Three.js (r128)** umgesetzt und benötigt einen modernen Webbrowser mit WebGL-Unterstützung. Das Modell ist für Desktop-Geräte optimiert, funktioniert aber auch auf Tablets und Smartphones.

## Forschungsfrage

> *Welche Faktoren beeinflussen die Wahrnehmung von Kosten, Nutzen und Vertrauen asynchroner Interview-Tools im Bewerbungsprozess aus Sicht von Rekrutierenden und Bewerbenden?*
